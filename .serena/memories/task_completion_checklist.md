# タスク完了時のチェックリスト

## コード変更後の確認事項

### 1. テスト実行
- 単体テストが通ることを確認
- デバッグ用スクリプトでの動作確認
- 既存機能への影響がないことを確認

### 2. アプリケーション動作確認
```bash
streamlit run app.py
```
- UIが正常に表示される
- 計算結果が期待通り
- エラーが発生しない

### 3. コード品質
- 型ヒントが適切
- docstringが記述されている
- 日本語コメントが分かりやすい
- エラーハンドリングが適切

### 4. Git管理
- 変更内容をコミット
- 適切なコミットメッセージ
- 必要に応じてブランチ作成

## 特に重要な確認項目
- 数値計算の精度・安定性
- 物理的な妥当性（単位、符号）
- UI表示との整合性
- メモリ・性能への影響
